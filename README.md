# yatranslator_bot
Telegram bot using Yandex translation API. Available as [@yatranslator_bot](https://t.me/yatranslator_bot)

Please use following commands to set language
```
/english
/spanish
/german
/french
/italian
/finnish
/chinese
/korean
/hebrew
/thai
/turkish
/swedish
/czech
/estonian
/latvian
/dutch
/arabic
/japanese
```
